# Spy-Cat-Agency-TEST
SCA project

To start the project you need to create .env file in main diretory by the .env_example and then run docker-compose by docker-compose up --build, 

If you want to start with test version then make in your .env VERSION=Test
If in prod than make VERSION=Prod

The difference is that in Prode there is JWT authorization.

You can see all endpoints at your_ip:your_host/swagger/
